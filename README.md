​A comprehensive analysis of global video game sales that converts raw CSV data into a professional PDF portfolio.
​Key Features:
​Data Integrity Check: Performed a mathematical audit to compare regional sales (NA, EU, JP, Other) against reported Global Sales, identifying discrepancies in the original dataset.
​Statistical Imputation: Used Median values for missing years to protect against the skewing effects of outliers.
​Visual Analytics: * Market Share: Created regional pie charts to visualize global dominance.
​Genre Trends: Generated horizontal bar charts to identify high-performing software categories.
​Automated Reporting: Built a custom PDF generator using the FPDF library that dynamically creates a multi-page technical report including pivot tables and embedded visualizations
